# Tencent RTC Conference

> Tencent RTC Conference (TUIRoomKit) is a multi-person audio/video conferencing solution. It provides a UIKit (low-code, drop-in UI) and a Core SDK (RoomEngine) for full customization. Conference supports Web, Android, iOS, Flutter, and Electron.

This index is the entry point for LLMs. It links to shared product-level docs and to one `llms.txt` per supported platform. Each linked page also has a `.md` twin (append `.md` to the URL) containing the raw markdown.
